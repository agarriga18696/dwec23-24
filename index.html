Unidad I

I.I Historia de JavaScript

JavaScript nace en 1995, creado por un señor que se llama Brendan Eich. Éste hombre trabajaba en una empresa de tecnología, en la cual usaba como navegador NetScape. Éste lenguaje de programación se implementó con mucha velocidad gracias a su éxito, y Microsoft sacó una versión propia llamada JScript. Llega un punto en que se estandarizó el lenguaje gracias a la implementación de ECMAScript, de ECMA International, y se publicó en 1997. A lo largo de los años se ha ido actualizando hasta su versión más reciente, ES 2015, también conocida como ES6 por ser la sexta versión.

Para poder interpretar el lenguaje los navegadores incluyen un motor de JavaScript, los más conocidos son:

  1. Spider Monkey
  2. V8
  3. JavaScriptCore
  4. Carakan
  5. Chakra

JavaScript es un lenguaje single thread, solo tiene un hilo de ejecución, esto quiere decir que las tareas se ejecutan secuencialmente. En 2008 aparece el motor V8, desarrollado por Google, supone una gran mejora entre los motores ya existentes y es una mezcla entre compilador e intérprete. Esta técnica se llama JIT (Just-In-Time Compiler). El resto de motores utilizan ésta técnica.

El motor traduce el código a bytecode, que se hace lo más rápido posible. Éste bytecode lo ejecuta el intérprete. Éste sistema de intérprete-compilador tiene una característica llamada Profiler, que es un mecanismo para optimizar el código, con el fin de mejorar el rendimiento.

Hay una serie de elementos fundamentales que necesita JavaScript para su correcto funcionamiento:

  1. Call Stack Heap
  2. Event Loop & Callback Queue
  3. Web Apis

Estas APIS facilitan la implementación de algunas funcionalidades, algunas de las más conocidas son:

  1. DOM
  2. XML HTTP Request
  3. Geolocalización
  4. Funciones timer
